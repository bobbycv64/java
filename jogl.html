Java Open Graphics Language (JOGL)

Home Page:  https://jogl.dev.java.net/
Download the software:  Precompiled binaries and documentation
  • Here are the latest releases:  https://jogl.dev.java.net/servlets/ProjectDocumentList, click on the release you wish to obtain.
  • You must download both jogl.jar and which platform you wish to execute, e.g. jogl-natives-win32.jar
Installation Notes (Windows)

Sometimes there are issues with the libraries, do the following and you will not run into problems.  There are two dll files required for the Windows operating system.  This is easily fixed by copying two dll files to the windows32 directory, e.g.:

copy <location of jdk>\jre\bin\awt.dll   C:\WINDOWS\system32\awt.dll
copy <location of jdk>\jre\bin\jawt.dll  C:\WINDOWS\system32\jawt.dll

Copy the two jar files, e.g. jogl.jar, jogl-natives-win32.jar and supporting jar files as suggested by the installation notes to the C:\WINDOWS\system32\ directory.


Extract the jogl.dll and jogl_cg.dll files from the jogl-natives-win32.jar and place in the C:\WINDOWS\system32\ directory.

Classpath Settings

set classpath=.;%classpath%;C:\WINDOWS\system32\jogl.jar   <- don't forget the ".", very important.

Import the following code to make sure that JOGL has been installed and implemented correctly:
import net.java.games.jogl.*;

public class HelloJOGL {
  public static void main (String args[]) {
    try {
        System.loadLibrary("jogl");
        System.out.println("Hello JOGL! (The native libraries are installed.)");
        GLCapabilities caps = new GLCapabilities();
        System.out.println("Hello JOGL! (The jar appears to be available.)");
    } catch (Exception exception) {
        System.out.println(exception.getMessage());
        System.out.println(exception);
    }
  }
}

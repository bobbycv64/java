Remote Method Invocation (RMI)

RMI enables Java programs to communicate with Java distributed objects. There are two types of RMI programs.  RMI-JRMP, Java to Java and RMI-IIOP, any object-oriented language to Java.  To get additional information go to http://java.sun.com

Below is a very basic example of how to execute RMI.  Just copy all files to your system and execute the batch file and you should see the sparks fly.
 
Client Java Program RMIClient.java
Interface RMICommon.java
Server Java Program RMIServer.java
Windows make and execute file run.bat
Policy File client.policy

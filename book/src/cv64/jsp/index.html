Java Server Pages (JSP)

JSP is yet another great product offering in the Java family.

However let me give you good advise.

JSP likes to keep information in cache and nearly impossible to remove.  One trick is to recompile a Servlet and this will refresh your JSP server.  Another trick is to actually make a change in a JSP file and save the file.  Just doing a save won't do the trick because the timestamp hasn't been modified.  Yet another trick is to delete the files in the Tomcat work directory, the location of the cached files.
